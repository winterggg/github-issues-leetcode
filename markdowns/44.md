# 问题

https://leetcode.cn/problems/binary-search/

# 笔记

板子题，`while l < r:` 是最佳实践~

# 代码

## Python

```python
class Solution:
    def search(self, nums: List[int], target: int) -> int:
        l, r = 0, len(nums) - 1
        while l < r:
            mi = l + r >> 1
            if nums[mi] >= target:
                r = mi
            else: l = mi + 1
        
        return l if nums[l] == target else -1
```
