# 问题

https://leetcode.cn/problems/two-sum/

# 笔记

mappings 优化，减少一次循环

# 代码

## Python

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        mapping = {}

        for i, v in enumerate(nums):
            t = target - nums[i]
            if t in mapping:
                return [mapping[t], i]
            mapping[v] = i
        
        return []
```
