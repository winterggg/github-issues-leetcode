# 问题

https://leetcode.cn/problems/sqrtx/

# 笔记

经典二分，二分目标：p * p <= x 的最大值，注意二分的死循环处理。

# 代码

## Python

```python
class Solution:
    def mySqrt(self, x: int) -> int:
        l, r = 0, x
        while l < r:
            mi = l + r + 1 >> 1
            if mi * mi <= x:
                l = mi
            else: r = mi - 1
        
        return l
```
